# AntyLogout ⚔️

A lightweight and high-performance anti-combat-log plugin for modern Minecraft servers (1.21.4+).

## 🌟 Features
*   **BossBar Timer:** A visible red countdown bar at the top of the screen.
*   **Logout Punishment:** Automatically kills players who leave the server during combat to prevent "combat logging".
*   **Modern API:** Built specifically for **Paper/Spigot 1.21.4** using **Java 21**.
*   **Bypass Permission:** Admins can be excluded using the `antylogout.bypass` permission.

## 🛠️ Installation
1. Download the latest `AntyLogout.jar` from the Modrinth page.
2. Drop it into your server's `plugins` folder.
3. Restart your server.

## ⚙️ Requirements
*   **Server:** Paper, Spigot, or Purpur (1.21.x).
*   **Java:** Version 21 or higher.
